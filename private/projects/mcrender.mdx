# mcrender

A Minecraft thumbnail utility that I scaled to **15,000 monthly active users** and made completely self-sustainable through strategic monetization.

![mcrender dashboard](/images/remoteupload-usage.jpeg)

## Overview

mcrender started as a simple tool to help Minecraft content creators generate professional thumbnails. What began as a side project evolved into a full-fledged SaaS platform serving thousands of creators worldwide.

## The Journey

### Problem

Minecraft YouTubers and content creators needed an easy way to create eye-catching thumbnails without complex 3D software. Existing solutions were either too complicated or didn't provide the customization options creators needed.

### Solution

I built mcrender to bridge this gap - a web-based tool that makes it simple to:
- Generate high-quality Minecraft character renders
- Customize poses, skins, and backgrounds
- Export production-ready thumbnails instantly

## Key Achievements

- **15K Monthly Users**: Grew from 0 to 15,000 active users through organic growth and word-of-mouth
- **Self-Sustainable**: Implemented a freemium model with Polar integration that covers all operational costs
- **99.9% Uptime**: Maintained reliable service with optimized infrastructure
- **Global Reach**: Users from over 50 countries creating thumbnails daily

## Technical Stack

The platform is built with modern web technologies for performance and scalability:

- **Next.js 14** - Server-side rendering and API routes
- **Three.js** - 3D rendering engine for real-time previews
- **React** - Component-based UI architecture
- **Polar** - Payment processing and subscription management
- **Vercel** - Deployment and edge functions

## Features

### Free Tier
- Basic character rendering
- Standard poses
- Watermarked exports
- 720p resolution

### Premium Tiers
- Custom pose editor
- High-resolution exports (up to 4K)
- No watermarks
- Priority rendering queue
- Advanced lighting controls

## Learnings

Building and scaling mcrender taught me invaluable lessons:

1. **User Feedback is Gold**: The most successful features came directly from user requests
2. **Monetization Matters**: Finding the right pricing model took iteration but was crucial for sustainability
3. **Performance is Key**: Optimizing 3D rendering for web was challenging but essential for user experience
4. **Community Building**: Engaging with users on Discord and Twitter drove organic growth

## Impact

mcrender has helped thousands of creators improve their content quality and grow their channels. Seeing users succeed with the tool they built with mcrender is incredibly rewarding.
